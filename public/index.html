Инструкция по работе с программой:

• Для того, чтобы прикрепить грузик на коромысло (или подвесить его на пружинку), нажмите на грузик, а затем на позицию, куда хотите перетащить. При этом диск должен находиться в исходном положении (0 градусов).

• Для того, чтобы повернуть диск, переместите ползунок (красный квадратик под установкой) на нужную длину.

• Для того, чтобы запустить диск, отклоните его и нажмите "СТАРТ".

• Для того, чтобы остановить секундомер, нажмите "СТОП".

• Для того, чтобы обнулить секундомер, остановить диск и вернуть грузики в исходное положение, нажмите "СБРОС".
